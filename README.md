# Six-Dimensional-Data-Quality-Color-Framework
A standardized framework for evaluating data quality across six core dimensions, with integrated color-coded visualization for intuitive quality assessment.

## ✨ Key Features
- **6-Dimensional Data Quality Metrics**: Completeness, Accuracy, Consistency, Timeliness, Uniqueness, and Validity.
- **Color-Coded Visualization**: Intuitive red/yellow/green grading for quick quality status identification.
- **Modular Design**: Separated `code/` (core logic) and `docs/` (documentation) for maintainability.
- **Reproducible Environment**: `requirements.txt` for easy dependency setup.
